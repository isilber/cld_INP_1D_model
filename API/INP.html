INP module¶

This module include INP population class and its sub-classes for different PSDs. In addition, it includes the Jhet class.

Class to load Jhet LUT and assign c and m coefficient values based on requested INP type.

set the ABIFM linear fit coefficients for the INP type.

nucleus_type: str

INP type to use (must match the LUT (not case sensitive).

coeff_filename: str

path and filename of Jhet coefficients’ Table. By default using the values from Table 1 in Knopf and Alpert, 2013, DOI: 10.1039/C3FD00035D.
